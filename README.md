Django User Authentication System <br>
Overview <br>
This project is a simple web-based user authentication system built using Django. It allows users to create accounts, log in, access a protected home page, and log out. The application demonstrates core concepts of web development, authentication, and automated testing within a Django framework.
The system was developed as part of coursework focused on software quality assurance and deployment, with an emphasis on functionality, security basics, and test coverage. <br>
Features <br>
-User registration with username, email, and password validation <br>
-Secure login and logout functionality using Django’s authentication system <br>
-Protected home page accessible only to authenticated users <br>
-Input validation for password mismatch and incorrect login attempts <br>
-Automated test suite covering key user flows <br>
Technologies Used <br>
-Python <br>
-Django 3.0.3 <br>
-SQLite (default Django database) <br>
-HTML/CSS (with Bootstrap for styling) <br>
-Django Test Framework (unittest) <br>
